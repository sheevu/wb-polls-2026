# West Bengal Poll Predictions 2026

AI-Powered Interactive Dashboard by **Sudarshan AI Labs**

## Overview

Comprehensive district-wise analysis across all 294 assembly constituencies for the West Bengal Assembly Elections 2026.

## Features

- Live AI Projection
- Poll of Polls aggregation
- 3D Compass visualization
- Party-wise seat predictions
- Data Visualizer with D3 and React
- Infographic view

## Tech Stack

- HTML5, CSS3, JavaScript
- D3.js for data visualization
- Deployed on Vercel
